# React + TypeScript API Integration Demo

This project demonstrates how to integrate APIs in a React application using TypeScript. It showcases two distinct approaches:
1.  **Consuming an External API:** Fetching data from a public API (JSONPlaceholder).
2.  **Full Stack Integration:** Creating and consuming a custom local Backend API using Node.js & Express.

-   **Type-Safe Data Fetching:** Uses TypeScript Interfaces (`User.ts`, `Todo.ts`) to ensure data reliability.
-   **React Router:** Single Page Application (SPA) navigation between components (`/users`, `/todos`).
-   **Dual Data Sources:**
    -   **Users Page:** Fetches from [JSONPlaceholder](https://jsonplaceholder.typicode.com).
    -   **Todos Page:** Fetches from a local custom Express server.
-   **Modern React:** Uses Functional Components, `useState`, and `useEffect`.
-   **Backend:** A lightweight Node.js/Express server with CORS enabled.
